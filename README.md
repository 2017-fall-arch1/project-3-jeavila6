# Project 3: LCD Game
## Pong
Players use the paddles to hit a ball back and forth. The goal is for each player to reach five points before the opponent. Points are earned when one fails to return the ball to the other.

## Controls
* S1 - P1 move left
* S2 - P1 move right
* S3 - P2 move left
* S4 - P2 move right

## Play
In 'game' directory,
`$ make load`
